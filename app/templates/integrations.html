📖 API Reference

Use your API token to make requests to ContentForge.

# Generate content via API
curl -X POST https://contentforge.local/api/v1/generate \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"prompt": "Write a blog post about AI", "provider": "gemini"}'
# List your generations
curl -H "Authorization: Bearer YOUR_TOKEN" \
  https://contentforge.local/api/v1/generations
